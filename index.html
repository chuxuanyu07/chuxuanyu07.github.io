前言

正题

拉格朗日插值

众所周知，一个 nnn 次多项式函数可以被 n+1n + 1n+1 个点唯一确定。

那么现在给定 n+1n + 1n+1 个点：

(x0,y0),(x1,y1)......,(xn,yn)(x_0,y_0),(x_1,y_1)......,(x_n,y_n) (x0​,y0​),(x1​,y1​)......,(xn​,yn​)

显然需要满足条件 ：

∀i≠j,xi≠xj\forall i \ne j,x_i \ne x_j ∀i=j,xi​=xj​

（否则不是函数了）

显然我们可以通过高斯消元做到 O(n3)O(n^3)O(n3)，但是它太慢了…

所以我们需要拉格朗日插值，它可以做到 O(n2)O(n^2)O(n2) 的优秀复杂度。

我们现在想要 n+1n + 1n+1 个函数，第 iii 个 fi(x)f_i(x)fi​(x) 满足：在 x=xix = x_ix=xi​ 时，fi(x)=yif_i(x) = y_ifi​(x)=yi​ ，否则 fi(x)=0f_i(x) = 0fi​(x)=0。

则我们需要的函数显然为 ：

g(x)=∑i=0nfi(x)g(x) = \sum_{i = 0}^n f_i(x) g(x)=i=0∑n​fi​(x)

现在的关键就是构造所有的 fif_ifi​。

如果我们只是希望 ∀x≠xi,fi(x)=0\forall x \ne x_i,f_i(x) = 0∀x=xi​,fi​(x)=0 的话，显然有一个构造的方法:

fi(x)=∏j≠i(x−xj)f_i(x) = \prod_{j \ne i} {(x - x_j)} fi​(x)=j=i∏​(x−xj​)

这个应该很显然。

那么现在需要做的，就是让这个函数在 x=xix = x_ix=xi​ 时 fi(x)=yif_i(x) = y_ifi​(x)=yi​。

首先肯定乘上一个 yiy_iyi​。

fi(x)=∏j≠i(x−xj)×yif_i(x) = \prod_{j \ne i} {(x - x_j)} \times y_i fi​(x)=j=i∏​(x−xj​)×yi​

现在要做的，就是让 x=xix = x_ix=xi​ 时，yiy_iyi​ 乘上的是个 111。

也就是我们需要考虑，当 x=xix = x_ix=xi​ 的时候：

∏j≠i(x−xj)×?=1\prod_{j \ne i}(x-x_j) \times ? = 1 j=i∏​(x−xj​)×?=1

则我们可以乘上一个：

1∏j≠i(xi−xj)\frac{1}{\prod_{j \ne i}(x_i - x_j)} ∏j=i​(xi​−xj​)1​

那么 fif_ifi​ 就出来啦：

fi(x)=∏j≠i(x−xj)∏j≠i(xi−xj)×yif_i(x) = \frac{\prod_{j \ne i}(x-x_j)}{\prod_{j \ne i}(x_i - x_j)}\times y_i fi​(x)=∏j=i​(xi​−xj​)∏j=i​(x−xj​)​×yi​

我们需要的函数 g(x)g(x)g(x):

g(x)=∑i=0nfi(x)g(x) = \sum_{i = 0}^n f_i(x) g(x)=i=0∑n​fi​(x)

写在一起就是:

g(x)=∑i=0n(yi×∏j≠i(x−xj)(xi−xj))g(x) = \sum_{i = 0}^n (y_i \times\prod_{j \ne i}{\frac{(x-x_j)}{(x_i - x_j)})} g(x)=i=0∑n​(yi​×j=i∏​(xi​−xj​)(x−xj​)​)

对于特殊点的优化

如果这些点满足 xi=ix_i = ixi​=i ，那么这个函数可以写成：

g(x)=∑i=0n(yi×∏j≠i(x−j)(i−j))g(x) = \sum_{i = 0}^n (y_i \times \prod_{j \ne i}{\frac{(x-j)}{(i - j)})} g(x)=i=0∑n​(yi​×j=i∏​(i−j)(x−j)​)

考虑优化掉 ∏j≠i(x−j)(i−j)\prod_{j \ne i}{\frac{(x-j)}{(i - j)}}∏j=i​(i−j)(x−j)​ 的这个 O(n)O(n)O(n)。

假设我们现在要求的是 g(k)g(k)g(k)，处理分子可以设置数组:

重心拉格朗日插值

终于记起来把最后一点点补上了 同时，如果我们需要支持动态插入一个点，然后询问某一个单点的值时，每次暴力做都需要 O(k2)\mathcal{O}(k^2)O(k2) 的时间，不太划算，所以考虑使用 重心拉格朗日插值法。
